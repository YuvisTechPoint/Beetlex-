# BeetleX Reference Repositories

Curated open-source projects referenced during the BeetleX hackathon platform build:

| Area | Repository | Why |
|------|------------|-----|
| UI components | [shadcn/ui](https://github.com/shadcn-ui/ui) | Accessible primitives + Tailwind |
| Mock API | [MSW](https://github.com/mswjs/msw) | Network-level API interception |
| Server state | [TanStack Query](https://github.com/TanStack/query) | Caching, polling, invalidation |
| Client state | [Zustand](https://github.com/pmndrs/zustand) | Lightweight stores for auth/UI/notifications |
| Routing | [React Router](https://github.com/remix-run/react-router) | SPA routing + protected routes |
| Forms | [React Hook Form](https://github.com/react-hook-form/react-hook-form) + [Zod](https://github.com/colinhacks/zod) | Validation |
| Charts | [Recharts](https://github.com/recharts/recharts) | Organizer dashboard analytics |
| Toasts | [Sonner](https://github.com/emilkowalski/sonner) | Notification toasts |

See `SYSTEM_DESIGN.md` for architecture decisions and `GAP_AUDIT.md` for implementation status.
